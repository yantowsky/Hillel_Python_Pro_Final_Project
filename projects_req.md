1. **Розробка двох Django проектів**:
   Створити два окремих Django проекти - "ProjectA" та "ProjectB". Один із проектів може бути написано на React.

2. **REST API**:
   Реалізувати REST API в кожному проекті з використанням Django REST Framework. API повинно забезпечувати створення,
   оновлення, видалення та отримання даних.

3. **Комунікація через API**:
   Налаштувати взаємодію між "ProjectA" та "ProjectB" через їхні REST API. Здійснювати обмін даними між проектами.

4. **Docker**:
   Контейнеризувати кожний Django проект з використанням Docker. Забезпечити легку розгортаність та масштабованість за
   допомогою Docker-контейнерів.

5. **Тести**:
   Написати тести для кожного Django проекту, включаючи модульні, функціональні та інтеграційні тести. Перевірити роботу
   REST API, а також коректність взаємодії між проектами через API.

6. **Таски**:
   Використовувати Celery для асинхронного виконання тасків в кожному проекті. Реалізувати асинхронні функції, такі як
   обчислення або відправка повідомлень.

7. **Class Based Views та наслідування**:
   Використовувати Class Based Views для створення різних сторінок та функціональності в кожному Django проекті.
   Використати наслідування для спадкового перевизначення функціональності стандартних класів.

8. **Перевизначення моделі користувача та пермішини**:
   Розширити вбудовану модель користувача Django або створити власну модель користувача з додатковими полями в кожному
   проекті. Реалізувати систему пермішин для керування доступом користувачів до різних частин проектів.

9. **Bootstrap**:
   Використовуйте фреймворк Bootstrap для створення стилізованого та адаптивного інтерфейсу вашого Django додатку.
   Використовуйте готові компоненти, сітку та CSS-стилі для швидкого та ефективного розроблення користувацького
   інтерфейсу.

**Теми до дипломного проекту:**

1. **Електронний журнал для шкіл на Django (React) з можливістю відстеження успішності учнів та спілкування з батьками.**

   Опис вимог:
    - Додати систему обліку відвідування та оцінок для кожного учня.
    - Розробити інтерфейс для батьків, де вони можуть переглядати звіти про успішність своїх дітей та спілкуватися з
      вчителями.
    - Забезпечити можливість планування та відстеження домашніх завдань для учнів.

2. **Організатор подій та реєстраційна платформа на Django (React) для проведення конференцій, семінарів та інших подій.**

   Опис вимог:
    - Додати можливість створення різних типів подій (конференції, семінари, воркшопи тощо) з різними параметрами
      реєстрації.
    - Розробити систему керування спікерами та програмою події.
    - Забезпечити можливість генерації та надсилання електронних квитків учасникам.

3. **Онлайн-маркетплейс для продажу товарів на Django (React).**

   Опис вимог:
    - Додати систему оцінок та відгуків для товарів та продавців.
    - Реалізувати систему рекомендаційних товарів на основі попередніх покупок користувачів.
    - Додати можливість організації різних акцій та знижок для товарів.

4. **Платформа для організації та ведення онлайн-курсів на Django (React).**

   Опис вимог:
    - Розробити систему тестування та автоматичної оцінки результатів студентів.
    - Додати можливість проведення вебінарів та відеоуроків в режимі реального часу.
    - Забезпечити можливість індивідуальної консультації для студентів з викладачами.

5. **Організація онлайн-консультацій та спілкування з пацієнтами у медичних установах на Django (React).**

   Опис вимог:
    - Реалізувати систему обліку медичних записів та історії хвороб для кожного пацієнта.
    - Додати можливість спілкування з лікарем через текстові повідомлення або відеоконсультації.
    - Забезпечити можливість планування та запису на прийоми онлайн.
